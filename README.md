# ldoh-ldc-reward

为 LDOH 公益站提供增强功能。

## 脚本列表

- `ldoh-ldc-reward.user.js`: 为 LDOH 公益站添加 LDC 打赏功能。
- `ldoh-new-api-helper.user.js`: 为 LDOH 公益站提供余额查询、签到状态、密钥及模型查询功能。
